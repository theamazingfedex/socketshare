Socketshare

Create a temporary fileserver to share files over the internet.
download .ZIPdownload .TGZ

SocketShare

Install

npm install --global socketshare

Usage

After installation, simply execute

socketshare

from the command line to share the current folder over the network. Socketshare defaults to sharing the current directory over port 3000.

Alternate Usage

socketshare --port DESIRED_PORT --dir 'C:/My/Folder/To/Share'
